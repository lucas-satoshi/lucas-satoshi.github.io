---

## ⚡ Funcionalidades e Diferenciais do Projeto

* 📱 **Abordagem Mobile-First:** Interface totalmente responsiva, projetada e testada minuciosamente para telas desde smartphones (360px) até desktops ultra-wide (1280px+). Zero quebras de layout e sem barras de rolagem horizontal indesejadas!
* 🏗️ **Arquitetura 100% Semântica:** Código estruturado utilizando os padrões modernos do HTML5 (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`), otimizando o SEO básico e garantindo acessibilidade para leitores de tela.
* 🎨 **Visual Moderno e Fluido:** Design escuro (*Dark Mode*) sofisticado com acentuações em azul elétrico, utilizando tipografia profissional carregada via Google Fonts (*Oswald* e *Inter*).
* 🚀 **CSS Puro e Limpo:** Estilização desenvolvida do zero sem o uso de frameworks (como Tailwind ou Bootstrap), demonstrando domínio completo de *Box Model*, *Flexbox* e *CSS Grid*.
* 📬 **Formulário de Contato Ativo:** Canal direto operacional conectado de forma limpa ao serviço Formspree para recebimento de propostas e parcerias.
* ♿ **Acessibilidade Básica:** Todas as imagens possuem atributos `alt` descritivos e a paleta de cores foi validada para garantir ótimo contraste de leitura.
